These notes cover the early setup work for the website foundation. The focus was on keeping the project understandable, static-first, and ready for future content without adding unnecessary tools.

The setup introduced a small page structure, a shared base layout, and Markdown content collections for Projects and Lab. This creates a simple path for adding future entries by writing Markdown files instead of changing page markup each time.

The next step is to continue refining content slowly while keeping the technical structure stable.
