This project is the first foundation for a personal website. The goal is to keep the site small, static-first, and easy to maintain while leaving room for curated projects and lighter experimental notes.

The current version focuses on structure rather than visual complexity. It includes the main pages, a shared layout, basic navigation, and Markdown-based content collections for Projects and Lab.

Future updates can improve the content and design gradually without changing the core setup.
